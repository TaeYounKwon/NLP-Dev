Recurrent Neural Network(RNN) 요약

순환 신경망 RNN이란?
- 입력과 출력을 시퀀스 단위로 처리하는 시퀀스(Sequence)모델
- 앞서 배운 신경망들과 다른점?
-> 은닉층의 노드에서 활성화 함수를 통해 나온 결과값을 출력층 방향으로 보내면서
-> 다시 은닉층 노드의 다음 계산의 입력으로 보내는 특징

RNN에서 Cell이란?
- 은닉층 내부에서 활성화 함수를 통해 결과를 내보내는 역할을 하는 노드
- 이전의 값을 기억하려고 하는 일종의 메모리 역할
- 메모리 셀 혹은 RNN 셀이라고 표현
- 각각의 시점에서 바로 이전 시점에서의 은닉층의 메모리 셀에서 나온 값을 자신의 입력값으로 사용
- 메모리 셀에서 출력층 방향 또는 시점인 t+1의 자신에게 보내는 값을 은닉 상태(hidden state)라고 부름
- t시점의 메모리 셀은 t-1시점의 메모리 셀이 보낸 은닉 상태값을 t시점의 은닉상태 계산을 위한 입력값으로 사용

RNN의 벡터 단위
- 피드 포워드의 신경망에서는 뉴런이라는 단위를 썼다면
- RNN에서는 각각 입력 벡터, 출력 벡터, 은닉층의 은닉 상태라는 표현을 주로 사용
- RNN은 입력과 출력의 길이를 다르게 설계 할 수 있음
-> 한개의 입력, 여러개의 출력
--> 예) 하나의 이미지 입력에 대해서 사진의 제목을 출력하는 이미지 캡셔닝 작업

-> 여러개의 입력, 한개의 출력
--> 예) 입력 문서가 긍정인지 부정인지 판별하는 감성분류
--> 예) 스팸 메일 분류 등

-> 여러개의 입력, 여러개의 출력
--> 사용자의 문장을 입력하면 대답 문장을 출력하는 챗봇
--> 입력 문장을 변역하는 번역기
--> 태깅 작업 등

RNN의 수식
- https://wikidocs.net/22886문서 확인
- 은닉 상태값을 계산하기위해 하이퍼볼릭탄젠트 함수가 사용됨
- 출력층 결과인 Yt를 계산하기 위한 활성화 함수는 전의 과정처럼 케이스(이진분류, 다중 클래스 분류)에 따라 다름

깊은 순환 신경망(Deep Recurrent Neural Network)
- RNN에서 다수의 은닉층을 가진 것
- 순환 신경망에서 은닉층이 1개 더 추가되어 은닉층이 2개인 깊은(Deep) 순환 신경망

양방향 순환 신경망(Bidirectional Recurrent Neural Network)
- 빈칸 채워기 문제처럼 앞의 단어뿐만 아니라 뒤의 단어(미래의 값)까지 확인해 봐야 하는 경우 사용
- 단순이 입력 이전의 시점뿐만 아니라 입력 이후의 시점의 입력 또한 예측에 기여하는것
- 하나의 출력값을 예측하기 위해 기본적으로 두개의 메모리 셀을 사용
-> 첫번째 메모리 셀은 앞에서 배운것처럼 앞 시점의 은닉 상태(Forward States)를 전달받아 현재의 은닉 상태 계산
-> 앞 시점의 은닉상태가 아니라 뒤 시점의 은닉 상태(Backward States)를 전달 받아 현재의 은닉 상태를 계싼
