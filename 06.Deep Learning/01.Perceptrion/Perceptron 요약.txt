퍼셉트론(Perceptron) 요약

1. 퍼셉트론이란?
- 초기 형태의 인공 신경망으로 다수의 입력으로부터 하나의 결과를 내보내는 알고리즘
- 실제 뇌를 구성하는 신경 세포 뉴런의 동작과 유사
- x1, x2, x3등 다수의 입력값에 w 가중치 값을 받아 y 출력값을 내보내는것
- 실제 신경 세포 뉴런에서의 신호를 전달하는 축삭돌기의 역할을 퍼셉트론에서는 가중치가 대신함
- 각각 입력값에는 각각의 가중치가 존재
-> 가중치가 높다는 말은, 입력 값이 중요하다는 말
- 각 입력값이 가중치와 곱해져서 인공 뉴런에 보내지고, 그 값이 임계치(threshold)를 넘으면
-> 종착지에 있는 인공 뉴런은 출력 신호로 1을 내보냄
-> 아닐 경우는 0
-> 이러한 함수를 계단 함수라고 함

- 이때의 임계치를 좌변으로 넘기고 편향 b(bias)로 표현할 수도 있음
- 보통 그림에서는 1로 고정되고, 편향 b가 w처럼 곱해지는 변수로 표시됨

- 이렇게 뉴런에서 출력값을 변경시키는 함수를 활성화 함수(Activation Function)라고 함.
- 초기에는 계단 함수를 많이 사용, 이제 발전된 신경망들은 계단 함수 외에도 여러 다양한 활성화 함수 사용
- 시그모이드, 소프트맥스또한 활성화 함수 중 하나

- 앞서 배운 로지스틱회귀 모델이 인공 신경망에서는 하나의 인공 뉴런으로 볼 수 있음

단층 퍼셉트론(Single-Layer Perceptron)이란?
- 값을 보내는 단계와 값을 받아서 출력하는 두 단계로만 이루어진 퍼셉트론
- 각 단계를 층(layer)라 부르고, 두개의 층을 입력층(input layer)와 출력층(output layer)라고 함
- 컴퓨터는 두개의 값 0과 1을 입력해 하나의 값을 출력하는 회로가 모여 만들어지는데
-> 이 회로를 게이트(gate)라고 부름. 
- 초기에는 단층 퍼셉트론으로 NAND, OR, AND는 계산 및 구현 할 수 있어도
- XOR 게이트를 구현할 수 없었음

다층 퍼셉트론(MultiLayer Perceptron, MLP)이란?
- 입력층과 출력층 사이에 층을 더 추가한 것.
- 기존 AND, NAND, OR게이트에 입력층과 출력층만 존재했다면, 
- XOR은 입력, 출력 사이 AND, NAND, OR을 쌓아 XOR을 만들어 낸 것.
- 은닉층이 2개 이상인 신경망을 심층 신경망(Deep Neural Network,DNN)라고 함
