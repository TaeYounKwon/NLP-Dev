역전파(Back Propagation)이란?

먼저 인공신경망의 이해
- 인공신경망은 3개의 층 구조: 입력층, 은닉층, 출력층 
- 먼저 순전파대로 계산을 진행
-> 진행 방법은 전에 배워던 y= (w1x1+w2x2+....)+b
-> y값을 계산해준 후, 활성화함수(예: 시그모이드)에 넣어서 뉴런층 값을 계산
-> 출력층까지 위의 과정을 반복해서 진행
-> 출력층에 도착해서 나온 각 출력값(예측값)과 실제값에 대한 오차를 계산
--> 이때 손실함수(MSE 등)를사용 
-> 각 오차를 모두 더하면 전체 오차 E(total)이 됨

이제 역전파를 이해
- 순전파가 입력층에서 출력층으로 향한다면...
- 역전파는 반대로 출력층에서 입력층 방향으로 계산하면서 가중치를 업데이트해감
- 출력층 바로 이전의 은닉층을 N층이라고 하였을 때
- 출력층과 N층 사이의 가중치를 업데이트하는 단계를 역전파 1단계 
- 역전파 1단계는 마지막층 w에 대해 E(total)을 미분
-> 수학적 풀이는 https://wikidocs.net/37406 참고
-> 마지막으로 각 과정을 미분해서 구해진 값을 경사하강법을 통해 가중치를 업데이트
-> 이때 하이퍼파라미터에 해당되는 학습률(learning rate)을 미분 결과값에 곱해
-> 기존 w값 - 학습률x미분결과값해서 w값을 업데이트 해줌
- 이렇게 해서 역전파를 1번 실행하면 E(total)이 감소하는걸 확인 해 볼 수 있음
