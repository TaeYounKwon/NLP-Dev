언어 모델 요약

언어 모델이란?
- 단어 시퀀스(문장)에 확률을 할당하는 모델
- 통계를 이용한 방법 (전통적인 방법), 인공 신경망을 이용한 방법 (GPT, BERT)
- 가장 자연스러운 단어 시퀀스를 찾아내는 모델
- 이전 단어들이 주어졌을 때 다음 단어를 예측하도록 하는것

언어 모델링
- 주어진 단어들로부터 아직 모르는 단어를 예측하는 작업

단어 시퀀스의 확률 할당
- 기계 번역, 오타 교정, 음성 인식 등의 작업에서 보다 활률을 할당하여 보다 적절한 문장을 판단
- 이런 통계학 혹은 인공지능을 통한 확률을 계산하여 검색 엔진, 자동완성 등에서 사용 가능함.
