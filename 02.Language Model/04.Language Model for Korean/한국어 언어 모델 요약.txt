한국어 언어 모델 요약

한국어에 언어 모델을 사용하기 힘든 이유

1. 한국어는 어순이 중요하지 않음 -> 다음 단어로 어떤 단어나 등장 가능
- 예) 
-> 잭은 대학원 랩실에서 공부를 합니다
-> 대학원 랩실에서 공부를 합니다.
-> 공부를 합니다. 대학원 랩실에서
-> 잭은 공부를 합니다. 대학원 랩실에서
- 단어 순서를 뒤죽박죽으로 바꿔놔도 한국어 의미는 전달이 되기에...
- 확률에 기반한 언어 모델이 제대로 다음 단어를 예측하기가 힘듬

2. 한국어는 교착어임
- 띄어쓰기 단위인 어절 단위로 토큰화르 할 경우 문장에서 발생가능한 단어의 수가 비약적으로 늘음
- 한국어의 조사(그-는, -가, -를, -의, -와 등)때문에 
-> 토큰화를 통해 접사나 조사 등을 분리하는 것이 중요화됨

3. 띄어쓰기가 제대로 지켜지지 않음
- 한국어는띄어쓰기를제대로지키지않아도의미가전달되기에토큰이제대로분리되지않은체훈련이되는경우가많음.
